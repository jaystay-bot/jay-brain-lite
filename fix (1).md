# FIX MODE (PRECISION)

## Goal
Fix ONE issue with minimal changes.

## Rules
- Fix ONLY the reported issue
- Do NOT refactor unrelated code
- Do NOT improve architecture
- Do NOT run tests unless asked

## Scope
- Max 2 files
- Minimal edits only

## Process
1. Identify root cause
2. Fix the issue
3. Stop

## Verify
- If needed, explain how to test locally

## STOP
Do not continue into testing, refactoring, or deployment.
